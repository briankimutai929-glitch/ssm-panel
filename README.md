# SSM Panel

A Social Media Marketing (SSM) Panel built with React, Node.js, Express, and MongoDB.

## Features

- User Login
- Dashboard
- Place Orders
- Services
- Add Funds
- API Support
- Admin Panel
- Support Tickets

## Tech Stack

Frontend:
- React
- Tailwind CSS

Backend:
- Node.js
- Express

Database:
- MongoDB

Authentication:
- JWT# ssm-panel
